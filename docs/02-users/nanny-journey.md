# Nanny User Journey

## Phase 1: Discovery & Signup

### How Nannies Find Us:
- Facebook posts from other nannies
- Word of mouth
- Google search (parents sharing profiles)

### Signup Process:
1. Click "Become a Nanny" on website
2. Create account (email + password)
3. Fill out comprehensive profile form:
   - Personal information (name, age, location)
   - Experience (years, age groups, specializations)
   - Qualifications (certifications, education)
   - Availability (days/times available)
   - Rate expectations
   - Bio/about me (raw text input)
   - Profile photo upload

## Phase 2: AI Profile Generation

**What Happens**:
- AI processes raw form data
- Generates professional bio optimized for parent appeal
- Uses proven copywriting frameworks
- Creates compelling, trustworthy profile text

**Result**:
- Profile goes live immediately on platform
- Visible to all public browsers
- Professional presentation without nanny needing writing skills

**Status**: Profile Created (Tier 1) - Not yet verified

---

## Phase 3: Identity Verification

### Step 1: Passport Verification

**Process**:
1. Nanny uploads photo of passport
2. AI verifies:
   - Photo on passport matches profile photo
   - Name matches profile name
   - Passport is valid (not expired)
3. Manual admin review if AI uncertain

### Step 2: WWCC Verification

**Process**:
1. Nanny uploads Working With Children Check document
2. AI verifies:
   - Name matches profile
   - WWCC status is valid/current
   - Expiry date is future-dated
3. Manual admin review if needed

**Result**: Identity Verified (Tier 2)

**Unlocks**: Parents can now request interviews with this nanny

## Phase 5: Receiving Opportunities

### Interview Requests (After Tier 2 Verification)

**Trigger**: Parent requests interview with this nanny

**Process**:
1. Nanny receives email from Baby Bloom
2. Email shows:
   - Parent's name and brief info
   - 3 proposed interview times (parent selected these)
   - Request to select preferred time
3. Nanny clicks preferred time in email
4. Baby Bloom confirms with parent
5. Both parties receive calendar invite
6. Interview happens (video call, self-coordinated)

**Follow-up**:
- After interview, nanny receives friendly AI email asking how it went
- If nanny hired, Baby Bloom eventually asks parent if position filled
- Position closes automatically if filled or after extended time

---

### Babysitting Requests (After Tier 3 Verification)

**Trigger**: Parent posts babysitting request + nanny is one of 20 closest

**Process**:
1. Nanny receives email/SMS notification:
   - Date/time of babysitting job
   - Location (suburb only, not full address)
   - Number of children, ages
   - Rate offered
   - Special requirements
2. First nanny to accept gets the job
3. Parent's full address shared after parent ID verified (if first-time parent)
4. Job happens

**Important**: This is first-come-first-serve among 20 notified nannies
