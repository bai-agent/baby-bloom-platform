# Matching Rules

## Regular Nanny Matching

- Parent fills preference form → creates Open Nanny Position
- Algorithm matches nannies based on preferences
- Parent sees ranked list of matches
- Parent can request interviews

## Babysitting Matching

- Parent posts babysitting request
- System finds 20 closest Tier 3 verified nannies
- All 20 notified simultaneously (email/SMS)
- First to accept wins
- Address shared after parent ID verified (first time only)
